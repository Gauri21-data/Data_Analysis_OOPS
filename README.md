# Data_Analysis_OOPS
this project performs basic sales data analysis using Object-Oriented Programming (OOP)  Sales data is stored in dictionary format and analyzed using a custom class with multiple methods.

- OOP concepts in Python
- Sales calculations logic
- Clean and structured analysis code

## Technologies Used
- Python 
- Core Python (No external libraries)

---

# Dataset Description
The dataset contains:
- Product names
- Quantity sold
- Price per unit
